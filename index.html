Vad är HTML?

HTML eller Hyper Text Markup Language som det står för, är en standard/grundläggande byggsten för webben. HTML berättar för webläsaren hur innehållet ska visas grafiskt, strukturen på webbsidor. För övrigt är HTML ett märkspråk och inte programmeringsspråk, eftersom den strukturerar och läser inte av kommandon eller liknande.

Vad är Http?

Http betyder hypertext Transfer protocol och är ett kommunikationsprotokoll. Ett protokoll är regler/riktlinjer, och ett protokoll för kommunikationen innebär regler för kommunkationen mellan nätverk. Http används i webbläsare och hämtar innehåll från webbserverar, och med hjälp av http kan innehållet du ser på webbsidor bland annat visas upp.

Vad är Https?

Https betyder Hypertext transfer protocol secure. Både http och https hjälper användare få information från internet. Skillnaden är att https är säkrare. På sidor där privat information delas, t.ex mobilnummer och liknande är https extra viktigt. Https jobbar med protokollet Transport layer security, som krypterar känslig data och förhindrar att denna data läcks ut.

Vad är DNS?

DNS (Domain Name System) är en databas som är hierarkisk. Webbläsare använder sig av IP adresser, medan användare kommer åt webbsidor genom domännamn. Det DNS gör är då att översätta domäner till ip adresser. Med hjälp av DNS behöver vi inte memorera IP adresser, långa sifferkoder, istället kan vi memorera enkla domän som "youtube.com". DNS är utvecklad så att det snabbt ska gå att få fram vilken IP adress som är kopplat till vilket domännamn. Att den är hierarkisk innebär att informationen är organiserad i en trädstruktur som gör att det är lätt och snabbt att hitta rätt information.

Vad är CDN?

CDN (content delivery network) största uppgift är att hantera sättet ljud, bild och video (liknande innehåll) visas på webben och avlasta servern för att skapa en så liten fördröjning som möjligt, men ändå ha bra prestanda. Detta görs genom att flera serverar är ihopkopplade och hjälps åt. De är ihopkopplade till en globalt nät och specialiserade på att leverera innehåll.

Vad är Compressed data?

Compressed data är när data omkodas för att ta upp mindre plats på en disk eller snabbare skicka information på ett nätverk eller på internet. Det finns olika metoder för att genomföra detta, ett exempel är genom att koda om för att ta bort repitetativ data och byta ut det mot symboler.

Vad är Grid?

CSS Grid är underlättande för att skapa responsiva webblayouter. Det förenklar HTML och CSS genom att det är grid baserat/baserat på ett rutnät med rader och kolumner. Det byggs upp med ett parent element och flera child element, likt flexbox container och item.

Vad är WASM?

WASM - webassembly, är en ny typ av kod/teknologi som kan köras i webläsare. Wasm har stöd för flera olika språk som C, c++ och C#. Alltså kan man med hjälp av webbassemly köra C# applikationer direkt i webbläsaren, men det kan också arbeta tillsammans/integrera med javascript vilket underlättar för utvecklare då de kan ta del av flera bibliotek.

Vad är Chromium?

Chromium är en webbläsare med öppen källkod, som bland annat Google Chrome och Microsoft Edge är baserade på. Webbläsaren är lik google chrome, men annan logga och lite andra skillnader. För övrigt är chromium skapat av google.

Vad är svg?

svg betyder scalable vector graphic, och är ett XML baserat markup lanuage för två dimesionell vektorgrafik och stöder animation och interaktivitet. Det är textbaserat och jämfört med bilder i JPEG och PNG format så kan dessa bilder i SVG format bli renderade i vilken storlek som helst utan att bli sämre kvalitet.

Vad är png24?

Till och börja med är en PNG fil ett typ av bildformat. Skillnaden på vilket nummer som följer "png" är skillnaden hur många bitar de stöder. PNG8 stöder alltså 8 bitars färger och PNG24 24 bitars färger. PNG24 stöder också mycket fler färger, mer än 16 miljoner, till skillnad från PNG8's 256. PNG24 är alltså bättre för mer detaljerade bilder.
